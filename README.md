# SPS ASEAN Paragliding Quiz

An interactive, AI-guided paragliding knowledge assessment application built with Streamlit.

## Features

- **Interactive Quiz Interface**: Clean, user-friendly interface for taking the quiz
- **Two Proficiency Levels**: Beginner and Progressive pilot levels
- **AI Guidance**: Get AI-powered explanations and tips for each question
- **Certificate Generation**: Automatically generate HTML certificates upon completion
- **Progress Tracking**: Visual progress indicators and question navigation
- **Responsive Design**: Works on desktop and mobile devices
